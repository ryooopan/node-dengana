Dengana
=======

## What's this

東京の標準語を関西弁に自動翻訳するアプリケーションです。


## Installation

まず、Node.jsをインストールしていることが前提です。

```
git clone git@github.com:ryooopan/dengana.git
cd dengana
npm install
```

```
node app.js
```

## Examples

```
やられたらやり返す。 倍返しだ！
> やられたらやり返す。倍返しや！

きみの誠意を見せてもらおうじゃないか。
> あんさんの誠意を見せてもらおうやないか。

いや、違うんです。 私はわるくないんです。 半沢君が勝手に。。
> いや、ちゃうねん。 うちはわるくないねん。 半沢はんが勝手に。。
```
